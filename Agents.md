# AGENTS.md — Phase 0 + Phase 1: Environment Setup & STEP → Voxel Pipeline

## Project Overview

This is the foundation of a Rapid Process Planning (RPP) MVP system. The full
system ingests a STEP CAD file and a factory capability profile, then outputs a
rough process plan, machining time estimate, cost estimate, and job accept/reject
recommendation.

**Phase 0** sets up the project environment, installs dependencies, and
prepares training data.

**Phase 1** implements the geometry ingestion layer: STEP file → normalised
64³ voxel grid + structured metadata JSON. Every downstream phase depends on
this output being clean and consistent.

## 1.3 STEP Parsing — pythonOCC (Primary)

```python
def _parse_step_occ(step_path: str, stl_path: str) -> dict:
    """
    Parse STEP with pythonOCC.
    Returns raw geometry dict with keys:
      xmin, xmax, ymin, ymax, zmin, zmax  (in source units)
      volume, surface_area                (in source units)
      mesh_face_count
    Writes tessellated mesh to stl_path.
    """
```

Steps:

1. `reader = STEPControl_Reader()`
2. `status = reader.ReadFile(step_path)` — if status != `IFSelect_RetDone`
   raise `ValueError`
3. `reader.TransferRoots()`
4. `shape = reader.OneShape()` — if null raise `ValueError`
5. Bounding box:
   ```python
   bbox = Bnd_Box()
   brepbndlib.Add(shape, bbox)
   xmin, ymin, zmin, xmax, ymax, zmax = bbox.Get()
   ```
6. Volume:
   ```python
   props = GProp_GProps()
   brepgprop.VolumeProperties(shape, props)
   volume = props.Mass()
   ```
7. Surface area:
   ```python
   brepgprop.SurfaceProperties(shape, props)
   surface_area = props.Mass()
   ```
8. Tessellate and export STL:
   ```python
   BRepMesh_IncrementalMesh(shape, 0.1, False, 0.5, True).Perform()
   writer = StlAPI_Writer()
   writer.SetASCIIMode(False)
   writer.Write(shape, stl_path)
   ```
9. Count mesh faces via trimesh after writing:
   ```python
   m = trimesh.load(stl_path, force='mesh')
   face_count = len(m.faces)
   ```

## 1.4 STEP Parsing — cadquery (Fallback)

```python
def _parse_step_cadquery(step_path: str, stl_path: str) -> dict:
    """
    Parse STEP with cadquery fallback.
    Same return schema as _parse_step_occ.
    Volume and surface area are derived from the exported trimesh.
    """
    result = cq.importers.importStep(step_path)
    shape  = result.val()
    bb     = shape.BoundingBox()

    cq.exporters.export(result, stl_path)

    m = trimesh.load(stl_path, force='mesh')
    return {
        "xmin": bb.xmin, "xmax": bb.xmax,
        "ymin": bb.ymin, "ymax": bb.ymax,
        "zmin": bb.zmin, "zmax": bb.zmax,
        "volume":        m.volume,
        "surface_area":  m.area,
        "mesh_face_count": len(m.faces),
    }
```

## 1.5 Unit Detection and Conversion

Read the first 50 lines of the STEP file to detect the length unit:

```python
UNIT_MULTIPLIERS = {
    "MM": 1.0, "MILLIMETRE": 1.0, "MILLIMETER": 1.0,
    "M":  1000.0, "METRE": 1000.0, "METER": 1000.0,
    "IN": 25.4, "INCH": 25.4,
    "FT": 304.8, "FOOT": 304.8,
}

def _detect_unit_multiplier(step_path: str) -> tuple[float, str]:
    """
    Returns (multiplier, unit_name).
    Defaults to (1.0, "MM") if unit cannot be detected.
    """
    with open(step_path, "r", errors="ignore") as f:
        for i, line in enumerate(f):
            if i > 50:
                break
            upper = line.upper()
            for unit_key, mult in UNIT_MULTIPLIERS.items():
                if unit_key in upper:
                    return mult, unit_key
    return 1.0, "MM"   # default — add warning
```

Apply multiplier to all values from the parser:
- Linear dimensions (bounding box): multiply by `m`
- Areas: multiply by `m²`
- Volumes: multiply by `m³`

## 1.6 Voxelisation

```python
def _voxelise_mesh(stl_path: str, resolution: int) -> tuple[np.ndarray, bool]:
    """
    Convert STL mesh to solid-filled boolean voxel grid.

    Returns
    -------
    grid          : bool ndarray, shape (resolution, resolution, resolution)
    is_watertight : bool — True if mesh was watertight after optional repair
    """
    mesh = trimesh.load(stl_path, force='mesh')

    is_watertight = mesh.is_watertight
    if not is_watertight:
        trimesh.repair.fill_holes(mesh)
        trimesh.repair.fix_winding(mesh)
        trimesh.repair.fix_normals(mesh)
        is_watertight = mesh.is_watertight

    # Centre mesh at origin
    mesh.apply_translation(-mesh.bounds.mean(axis=0))

    # Scale so longest axis spans (resolution - 2) voxels
    # Leaves a 1-voxel empty border on each side
    longest = mesh.extents.max()
    if longest == 0:
        raise RuntimeError("Mesh has zero extent — degenerate geometry.")
    mesh.apply_scale((resolution - 2) / longest)

    # Voxelise and fill interior
    voxel_obj = mesh.voxelized(pitch=1.0).fill()
    grid = voxel_obj.matrix.astype(bool)

    # Pad or crop to exactly (resolution, resolution, resolution)
    grid = _pad_or_crop(grid, resolution)

    if grid.sum() == 0:
        raise RuntimeError("Voxelisation produced an empty grid.")

    return grid, is_watertight


def _pad_or_crop(grid: np.ndarray, target: int) -> np.ndarray:
    """Centre-pad or centre-crop a 3D bool array to (target, target, target)."""
    # Crop each axis if oversized
    for axis in range(3):
        if grid.shape[axis] > target:
            start = (grid.shape[axis] - target) // 2
            grid = np.take(grid, range(start, start + target), axis=axis)

    # Pad each axis if undersized
    out = np.zeros((target, target, target), dtype=bool)
    offsets = [(target - grid.shape[i]) // 2 for i in range(3)]
    slices  = tuple(slice(offsets[i], offsets[i] + grid.shape[i]) for i in range(3))
    out[slices] = grid
    return out
```

## 1.7 Raw Stock Calculation

```python
import math

def _compute_raw_stock(bbox_mm: dict, allowance: float) -> dict:
    """
    Apply stock allowance and round up to nearest 5 mm.

    Parameters
    ----------
    bbox_mm   : dict with keys "x", "y", "z"
    allowance : float e.g. 0.10 for 10%

    Returns
    -------
    dict with same keys, values rounded up to nearest 5 mm
    """
    def _up5(v: float) -> float:
        return math.ceil(v * (1.0 + allowance) / 5.0) * 5.0

    return {axis: _up5(dim) for axis, dim in bbox_mm.items()}
```

## 1.8 Metadata Schema

`metadata.json` written to `output_dir` must match this schema exactly:

```json
{
  "source_file":      "/abs/path/to/part.stp",
  "source_unit":      "MM",
  "resolution":       64,

  "bounding_box_mm":  { "x": 100.0, "y": 60.0, "z": 40.0 },
  "volume_mm3":       240000.0,
  "surface_area_mm2": 28800.0,
  "occupancy_ratio":  0.452,

  "raw_stock_mm":     { "x": 115.0, "y": 70.0, "z": 45.0 },

  "mesh_face_count":  1248,
  "mesh_is_watertight": true,

  "step_backend":     "occ",
  "voxel_file":       "/abs/path/to/voxel_64.npy",
  "mesh_file":        "/abs/path/to/mesh.stl",

  "warnings":         []
}
```

All file paths stored in metadata must be **absolute**.

Write atomically using a temp file + rename:

```python
import tempfile, os, json

def _write_json_atomic(data: dict, path: str) -> None:
    dir_ = os.path.dirname(os.path.abspath(path))
    with tempfile.NamedTemporaryFile(
        "w", dir=dir_, delete=False, suffix=".tmp"
    ) as f:
        json.dump(data, f, indent=2)
        tmp = f.name
    os.replace(tmp, path)
```

## 1.9 Full `process_step_file` Flow

```
process_step_file(step_path, output_dir, resolution, stock_allowance)
│
├─ 1.  Validate step_path exists            → FileNotFoundError if not
├─ 2.  os.makedirs(output_dir, exist_ok=True)
├─ 3.  _detect_unit_multiplier(step_path)   → (multiplier, unit_name)
├─ 4.  _parse_step_{backend}(step_path, mesh_stl_path)
│       └─ Apply multiplier to bbox/volume/area
├─ 5.  _voxelise_mesh(mesh_stl_path, resolution)
│       └─ Returns (grid, is_watertight)
├─ 6.  occupancy_ratio = grid.sum() / grid.size
├─ 7.  voxel_filename = f"voxel_{resolution}.npy"
│       np.save(voxel_path, grid)
├─ 8.  _compute_raw_stock(bbox_mm, stock_allowance)
├─ 9.  Assemble metadata dict
├─ 10. _write_json_atomic(metadata, metadata_path)
└─ 11. return metadata
```

# NOTES FOR CODEX

1. **Do not use binvox.** Use trimesh `.voxelized()` only. Binvox requires
   a platform binary and is not pip-installable.

2. **Always call `.fill()` after `.voxelized()`.** Without it the grid is
   a hollow surface shell, which breaks all downstream phases.

3. **Voxel filename includes resolution:** `voxel_{resolution}.npy` — not
   always `voxel_64.npy`. The test for default resolution checks for
   `voxel_64.npy`; the custom-resolution test checks for `voxel_32.npy`.

4. **All metadata paths must be absolute.** Use `os.path.abspath()` when
   assembling the metadata dict, not relative paths.

5. **Preserve `mesh.stl` always.** Phase 2 (feature recognition) and
   Phase 3 (setup analysis) reuse it.

6. **Unit conversion is critical.** STEP files from SolidWorks default to
   millimetres. FreeCAD exports may default to metres. A 1000× error in
   bounding box silently corrupts all downstream estimates.

7. **`warnings` must always be a list**, even when empty. Downstream code
   iterates over it unconditionally.

8. **Write metadata atomically** (temp file + `os.replace`). A partial
   write corrupts the pipeline for all downstream phases.
